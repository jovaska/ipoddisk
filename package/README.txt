To use iPodDisk:

1. drag iPodDisk.app to your Applications folder.
2. plugin iPod and make sure to enable disk mode. (iTunes > Preferences iPod Tab > General > Select Enable disk use)
3. after iPod drive icon shows up on your Desktop, launch iPodDisk.app.

For troubleshooting and other questions, please use our user forum:
http://www.nabble.com/iPodDisk-f13733.html
